# TicTacToe-CS

This repository contains a Tic-Tac-Toe game implemented in C#. 

## Description

TicTacToe-CS is a simple console-based implementation of the classic Tic-Tac-Toe game. Players take turns marking a space in a 3Ã—3 grid. The player who succeeds in placing three respective marks in a horizontal, vertical, or diagonal row wins the game.

## Features

- Two-player mode
- Simple and easy-to-understand code
- Console-based interface

## Requirements

- .NET SDK

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/Marijn-de-Mul/TicTacToe-CS.git
   ```
   
2. Navigate to the project directory:
```bash
cd TicTacToe-CS
```

3. Run the application:
```bash
dotnet run
```
